## 使用说明

在所有的 script 标签前使用 script 标签公开密钥：

```html
<script>
  window.xxx=1
</script>
```

## 可支持参数

1. POWER MODE 输入特效
  - `shake`： $0$ 或 $1$，输入时页面是否振动，默认关闭。
  - `colorful`： $0$ 或 $1$，输入特效是否彩色，默认关闭。
2. TITLE 标题变化
  - `leavetitle`：字符串，离开页面时变化的标题，默认 `w(ﾟДﾟ)w 不要走！再看看嘛！`。
  - `backtitle`：字符串，回到页面时变化的标题，默认 `♪(^∇^*) 欢迎肥来！`。
